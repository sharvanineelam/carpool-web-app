# CarPool – Project Overview

## Introduction
CarPool is a web based ride sharing platform that connects drivers who have empty seats in their vehicles with passengers traveling in the same direction.

The system allows drivers to publish ride details such as source, destination, available seats and ride price. Passengers can search for available rides and select rides that match their travel route.

## Objective
The main objective of CarPool is to simplify shared travel and improve transportation efficiency by connecting people traveling along similar routes.

The platform helps reduce transportation cost, traffic congestion and fuel consumption.

## Core Functionalities

User Authentication  
Users can create accounts and log into the system securely.

Ride Posting  
Drivers can publish rides with details like source, destination, seats and price.

Ride Search  
Passengers can search for available rides based on travel locations.

Ride Management  
Drivers can view and manage rides they have created.

User Profile  
Users can view their personal profile information and update profile images.

## System Workflow

User creates an account  
User logs into the platform  
Driver posts ride information  
Passenger searches available rides  
Driver manages rides from dashboard
